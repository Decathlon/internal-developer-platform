The Internal Developer Platform provides a comprehensive set of features to build your Internal Developer Platform. This section covers current capabilities and the roadmap for future development.

## Feature Overview

<div class="grid cards" markdown>

- 📊 **[Data Integration](data-integration.md)**

    ---

    Connect to any data source through Webhooks, Kafka, or Pub/Sub. Map incoming data to entities using JQ expressions.

    **Status:** 🕐 Planned

- 📈 **[Scorecards](scorecards.md)**

    ---

    Track engineering excellence with configurable levels, conditions, and weights. Assess your tech landscape's health.

    **Status:** 🕐 Planned

- 🚀 **[Self-Service Actions](self-service-actions.md)**

    ---

    Enable developers to execute complex workflows through simple, single-click actions with guardrails.

    **Status:** 🕐 Planned

</div>
